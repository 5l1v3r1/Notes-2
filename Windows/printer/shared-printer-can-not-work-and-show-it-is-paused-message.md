# Shared Printer Can not Work and Show "It is Paused" Message

## Problem
* Shared printer can not work and show "It is paused" message

## Solution
* Goto "Control Panel" -> "Printers" -> select the printer
* Right click the "Pause printing" option to make sure it's **UNCHECKED**
