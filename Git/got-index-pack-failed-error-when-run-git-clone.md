# Got "index-pack failed" Error when Run `git clone`

## Problem
* 1st `git clone` succeeded
* When run another `git clone`, got "index-pack failed" Error when Run `git clone`

## Root Cause
* Insufficient memory

## Solution
* Free some memory or restart PC and try again
